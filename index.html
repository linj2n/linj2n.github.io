Posted on 2018/02/07

什么是 EL

维护 JSP 页面中的 Java 代码 (Scriptlet) 是十分的不方便的，因此 Expressing Language (EL) 被设计成能够轻松地编写无脚本或不包含 Java 代码的 JSP 页面。也就是说使用 EL 编写的 JSP 页面可以不使用（包含）JSP 声明、 表达式或 Scriptlet。

EL 语法

以一个${开头，$}结束，表达式运算顺序从左向右，结果的类型强制为String，然后连接在一起。


EL 表达式若用于某个定制标签的属性值内，那么将会运算这个表达式，并强制结果字符串为该属性想要的类型。

对于expr-a[expr-b]的表达式，运算规则如下：

  • 从左向右，先运算 expr-a得到value-a
    • 若value-a为null，则返回null。
  • 运算expr-b得到value-b
    • 若value-b为null，则返回null。

根据value-a的类型，分别进行如下处理：

  • 若value-a的类型为java.util.Map，则检查value-b是否为Map中的一个键。
    • 若是，则返回value-a.get(value-b)
    • 若不是，则返回null
  • 若value-a的类型为java.util.List或数组，则强制value-b转换为一个int类型，强制失败会抛出异常。
    • Array类型，返回Array.get(value-a,value-b)
    • List类型，返回value-a.get(value-b)
  • 最后value-a只能是一个JavaBran，这种情况强制value-b为String。调用getter方法，返回结果。

其他 EL 运算符

除了[]和.运算符，EL 还提供几个其他的运算符：

  • 算术运算符
  • 关系运算符
  • 逻辑运算符
  • 条件运算符
  • empty运算符

EL 隐式对象

EL 通过一组隐式对象以访问各种对象。

对象 描述
pageContext 当前JSP页面的javax.servlet.jsp.PagetContext以获取request、response等隐式对象。
header 包含所有请求标头以标头为名称作为键的 Map
cookie 包含当前请求对象中所有Cookie对象的 Map
applicaScope 包含ServletContext对象中的所有属性并以属性名称作为键的 Map
sessionScope 包含HttpSession对象中所有属性并此属性名称作为键的 Map

概念

名称 解释 示例
标签 HTML是由一套标记标签 （markup tag）组成，通常就叫标签，标签由开始标签和结束标签组成
元素 元素指的是从开始标签到结束标签之间所有的代码，属性用来修饰标签的 <h1 align="center">居中标题</h1>
标题 标题会自动粗体，大写其中的内容，并且带有换行效果 <h1>标题1</h1>
段落 <p>标签即表示段落是paragraph的缩写，自带换行效果 <p>段落4 </p>
粗体 <b> <strong>b是bold的缩写，仅仅表示该文本是粗体的，并不暗示这段文字的重要性。strong虽然也是粗体，但是更多的是强调语义上的加重，提醒用户该文本的重要性。在SEO（搜索引擎优化）的时候，也更加容易帮助用户找到重点的内容 <b>，<strong>
斜体 <i>和<em>都可以表示斜体效果，i是italic的缩写，仅仅表示该文本是斜体的，并不暗示这段文字的重要性，em 是 Emphasized的缩写，虽然也是斜体，但是更多的是强调语义上的加重，提醒用户该文本的重要性。 常常用于引入新的术语的时候使用。
预格式 有时候，需要在网页上显示代码，比如java代码就需要用到<pre>
下划线 <ins>即下划线标签，但是通常来讲，不要给普通文本加下划线，因为用户会误以为是一个超链
删除线 <del>即删除标签，delete的缩写

位运算操作与其应用小结

Posted on 2018/01/25

位运算

在Java中，涉及到的位运算符如下

操作符 功能 例子
~ 按位求反 ~val
>> 带符号右移，相对与小数点整体向右移；正数高位补0，负数高位补1 val >> position，相当于val / (2^position)
<< 带符号左移，相对于小数点整体向左移；无论正负数，空出来的比特位都补0 val << position，相当于val * (2^position)
>>> 无符号右移，相对于小数点右移；无论正负数，高位补0
& 按位与 var1 & var2
竖线 按位或 var1 竖线 var2
^ 按位异或 var1 ^ var2
