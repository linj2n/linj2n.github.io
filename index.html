JavaScript 基础学习笔记

Posted on 2018/05/26

基础

  • JavaScript不区分整数与浮点数，统一用Number表示。
  • JavaScript 语言规定，对象的键名一律为字符串，所以，数组的键名其实也是字符串。
  • ==和===前者会自动转换数据类型再比较，后者不会，应该尽量使用===代替==。
  • NaN特殊的Number与所有其他值都不相等，包括它自己
  • JavaScript的数组可以包括任意数据类型。
  • 表达式不需要分号结尾。一旦在表达式后面添加分号，则 JavaScript 引擎就将表达式视为语句。语句和表达式的区别在于，前者主要为了进行某种操作，一般情况下不需要返回值；后者则是为了得到返回值，一定会返回一个值。

变量

  • JavaScript的变量可以是任意数据类型，使用var一个语句声明一个变量，并且变量名不能是JavaScript的关键字。
  • JavaScript是动态语言，因此一个变量虽然只能被var声明一次，但是可以使用=反复赋值，而且可以是不同类型的变量,如

字符串

  • 使用转义字符区分字符串中的‘与“
  • 反引号表示多行字符串
  • 模板字符串代替+
  • 常见的字符串操作

数组

  • 本质上，数组属于一种特殊的对象。typeof运算符会返回数组的类型是object。
  • 直接给Array的length赋一个新值会导致Array大小改变
  • 使用delete命令删除一个数组成员，会形成空位，并且不会影响length属性。
  • 如果是空位，使用数组的forEach方法、for…in结构、以及Object.keys方法进行遍历，空位都会被跳过。
  • 通过索引改变数组内元素，若索引超过length则会引起Array大小的改变
  • push和shift结合使用，就构成了“先进先出”的队列结构（queue）。

indexOf

slice

类似于String的substring()函数，功能是截取Array的部分元素，然后返回一个新的Array数组，依然是左右开区间截取数组。

push和pop

使用push()和pop()函数分别对数组再末尾添加若干元素，以及删除最后一个元素。

unshift和shift

unshift(...)表示向数组的头部添加若干元素，shift()表示删除数组的第一个元素。

sort

对当前的数组进行排序

reverse

翻转数组

splice

修改Array的万能方法，可以从指定的索引开始删除若干元素，然后从该位置添加若干元素。

concat

功能：将当前的数组和另外一个数组连接起来，返回一个新的数组。请注意，concat()方法并没有修改当前Array，而是返回了一个新的Array。

如果数组成员包括对象，concat方法返回当前数组的一个浅拷贝。所谓“浅拷贝”，指的是新数组拷贝的是对象的引用。

还可自动将Array才开，然后全部添加到新的Array里面。

join

功能：把当前Array的每个元素都用指定的字符串连接起来，然后返回连接后的字符串。

对象

JavaScript 的对象是一组键值组成的无序集合，通过.或['atrribute-name']操作符访问属性，但当属性名不是一个有效的变量名时，比如属性名包含特殊字符，则必须用‘’括起来，而且使用[]代替.

若访问一个不存在的属性不会报错，只会返回一个undefined。

添加会删除属性

判断一个属性是否是xiaoming自身拥有的，而不是继承得到的可以使用hasOwnProperty()方法。

Map 和 Set

定义初始化一个Map

Set与Map类似，是一组Key的组合，但不存储value。由于Key不能重复，所以在Set中没有重复的Key

Iterable

for … of

使用for ... of分别遍历数组、Map、Set

for ... of与for ... in的区别：后者遍历的实际是对象的属性名称，前者值遍历集合本身的元素。

forEach()

更好的遍历元素的方式是使用iterable内置的forEach方法，接受一个函数，每次迭代就自动回调该函数。

函数

JavaScript语言将函数视为一种值，与其他数据类型地位平等，因此被称为第一等公民。

三种声明函数的方法


你可以传递任意数量的参数给Function构造函数，只有最后一个参数会被当做函数体，如果只有一个参数，该参数就是函数体。这种方式是十分不直观，几乎无人使用。

与变量一样，一个函数可以被多次声明（赋值），后面的声明会覆盖前面的声明。

return语句
函数体内部的return语句，表示返回。特别注意JavaScript引擎遇到return语句会直接返回return后面的那个表达式的值。

不可以在条件语句中声明函数

属性、方法

Name

通过name获取参数函数的名字。

length

length属性返回函数预期传入的参数个数，即定义之中的参数个数。注意无论实际传入的参数的个数多少，但是length属性始终为2。

toString()

toString()方法返回一个字符串。

作用域

函数内部的变量提升
var命令声明的变量，无论在哪个位置，都会被提升到函数体的头部。

参数

  • 运行时，无论提供多少个参数，JavaScript都不会报错。若传入的参数数量下雨预期要求的数量，则省略的参数会被赋值为undefined。
  • 函数的参数的传递方式是传值传递(Pass by value)，
    • 若传递的值p是原始类型，则在函数内部，p的值是原始值的拷贝，无论如何修改，不会影响原始值。
    • 若传递的值obj是对象类型，则在函数内部，obj的值是原始值引用的拷贝，改变obj的属性，会影响到原始值，但如果obj = obj1，此时不会影响到原始值。

Arguments 对象

功能：
可以使用arguments对象在函数体内部读取所有的参数，严格模式下，arguments对象是一个制度对象，修改它是无效的。

与数组的关系：

  • 数组专用的方法如slice、forEach，不能再argument上使用。
  • 可以将arguments转换为真正的数组 **callee属性** `arguments`对象的`callee`属性，返回所对应的原函数。达到调用函数自身的目的，在严格模式里面是禁用的，不建议使用。

闭包(closure)

两个最大用处：
闭包是将函数内部和函数外部连接起来的一座桥梁。最大用处有二。一是可以读取函数内部的变量。另外一个是让这些变量始终保持在内存中。

其他的用处：
封装对象的私有属性和私有方法。

立即调用的函数表达式（IIFE）

eval 命令

操作DOM

DOM 是一个棵代表着 HTML 文档被浏览器解析后生成的树，操作一个 DOM 结点实际上有这么几个操作：

  • 更新：改变该DOM结点表示的HTML内容
  • 遍历：遍历该DOM结点下的子结点，以便下一步操作
  • 添加：在该DOM结点下增加一个子结点，相当于动态增加一个HTML结点
  • 删除：删除该DOM结点的内容以及它包含的所有子结点

获取 DOM

要想对 DOM 结点进行操作，前置操作是获取一个 DOM 结点。有两种方式获取 DOM 结点：

  • getElementByXXx()方法，XXx可以是id、tagName或className
  • querySelector()与querySelectorAll方法，需要了解selector语法规则

Servlet/JSP学习总结（6）Filter 过滤器

Posted on 2018/02/09

Filter 顾名思义为过滤器，无论客户端向服务端通过 URI 请求资源（JSP、Servlet、静态图片等），还是服务端响应客户端的请求，都必须经过实现了 Filter 接口的 filter 实例。

例如，在 filter 对象中通过HttpRequest.setCharacterEncoding()方法修改请求信息的编码方式。又如可以对请求的 URI 信息判别，使得请求转向其他的 JSP 页面，框架Struct 2 的控制器就是通过过滤器实现的。

总的来说，Filter 技术使得管理 Web 资源变得十分便利。不少 Java Web 高级技术都是建立在 Filter 之上，因此了解 Filter 这一种重要技术是十分必要的。
