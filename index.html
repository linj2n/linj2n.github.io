PreparedStatement

java.sql.PreparedStatement的优点：

  • Statement需要进行字符串拼接，可读性和维护性比较差；PreparedStatement使用参数设置，可读性好，不易犯错。
  • PreparedStatement有预编译机制，性能比Statement更快。
  • PreparedStatement防止SQL注入式攻击。
  • PreparedStatement避免重复编写SQL查询语句。

ORM

对象关系映射（Object Relational Mapping，简称ORM，或O/RM，或O/R mapping），是一种程序设计技术，用于实现面向对象编程语言里不同类型系统的数据之间的转换。

相比普通的JDBC编程，ORM有以下优点

  • 使用业务代码访问对象而不是数据库中的表
  • 从面向对象逻辑中隐藏 SQL 查询的细节
  • 实体是基于业务的概念而不是数据库的结构

ORM 一般包含四个实体

  • 一个 API 来在持久类的对象上实现基本的 CRUD 操作
  • 一个语言或 API 来指定引用类和属性的查询
  • 一个可配置的服务用来指定映射元数据
  • 一个技术和事务对象交互来执行 dirty checking, lazy association fetching 和其它优化的功能

Java 相关的 ORM 框架

  • Enterprise JavaBean(EJB)
  • Data Access Object (DAO)
  • Hibernate
  • myBaits

Servlet/JSP知识小结（6）Filter 过滤器

Posted on 2018/02/09

Filter 顾名思义为过滤器，无论客户端向服务端通过 URI 请求资源（JSP、Servlet、静态图片等），还是服务端响应客户端的请求，都必须经过实现了 Filter 接口的 filter 实例。

例如，在 filter 对象中通过HttpRequest.setCharacterEncoding()方法修改请求信息的编码方式。又如可以对请求的 URI 信息判别，使得请求转向其他的 JSP 页面，框架Struct 2 的控制器就是通过过滤器实现的。

总的来说，Filter 技术使得管理 Web 资源变得十分便利。不少 Java Web 高级技术都是建立在 Filter 之上，因此了解 Filter 这一种重要技术是十分必要的。

Servlet/JSP 知识小结（4） EL表达式

Posted on 2018/02/07

EL 全名为Expression Language。EL主要作用：

  1. 获取数据
    EL表达式主要用于替换JSP页面中的脚本表达式，以从各种类型的web域中检索java对象、获取数据。(某个 web 域中的对象，访问 Javabean 的属性、访问 List 集合、访问 Map 集合、访问数组)
  2. 执行运算
    利用EL表达式可以在JSP页面中执行一些基本的关系运算、逻辑运算和算术运算，以在JSP页面中完成一些简单的逻辑运算。${user==null}
  3. 获取web开发常用对象
    EL 表达式定义了一些隐式对象，利用这些隐式对象，web开发人员可以很轻松获得对web常用对象的引用，从而获得这些对象中的数据。
  4. 调用Java方法
    EL表达式允许用户开发自定义EL函数，以在JSP页面中通过EL表达式调用Java类的方法。

概念

名称 解释 示例
标签 HTML是由一套标记标签 （markup tag）组成，通常就叫标签，标签由开始标签和结束标签组成
元素 元素指的是从开始标签到结束标签之间所有的代码，属性用来修饰标签的 <h1 align="center">居中标题</h1>
标题 标题会自动粗体，大写其中的内容，并且带有换行效果 <h1>标题1</h1>

预格式 有时候，需要在网页上显示代码，比如java代码就需要用到<pre>
下划线 <ins>即下划线标签，但是通常来讲，不要给普通文本加下划线，因为用户会误以为是一个超链

位运算

在Java中，涉及到的位运算符如下

操作符 功能 例子
~ 按位求反 ~val
>> 带符号右移，相对与小数点整体向右移；正数高位补0，负数高位补1 val >> position，相当于val / (2^position)
<< 带符号左移，相对于小数点整体向左移；无论正负数，空出来的比特位都补0 val << position，相当于val * (2^position)
>>> 无符号右移，相对于小数点右移；无论正负数，高位补0
& 按位与 var1 & var2
竖线 按位或 var1 竖线 var2
^ 按位异或 var1 ^ var2
