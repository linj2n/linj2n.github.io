242.Valid Anagram

From：242.Valid Anagram

Description

Given two strings s and t, write a function to determine if t is an anagram of s.

For example,
s = "anagram", t = "nagaram", return true.
s = "rat", t ="car", return false.

Note:
You may assume the string contains only lowercase alphabets.

Follow up:
What if the inputs contain unicode characters? How would you adapt your solution to such case?

Input and output analysis

Approach#1 Sorting

Solution

对s、t进行排序，然后逐一检查s、t对应位置的元素是否相同，不相同则返回false

Code

Complexity

Time Space
O( nlogn ) O( n )

Approach#2 Use Map

Solution

由题意可知，当s、t包含的字母种类、同一个字母出现的次数相同，只是字符排列顺序不同，此时两个字符串构成Anagram。因此可以使用映射表记录字符串元素出现的个数，利用两字符串元素出现次数相同的性质很容易得出结果。

这里有两种判别的处理方法，第一种处理比较简单和笨拙，第二种更灵活、效率更高（类似于657. Judge Route Circle中的处理）：

  • 分别建立两个映射表，记录两个字符串中元素出现的个数，最后逐一对比两映射表对应键的值是否一一对应。
  • 只用一个哈希表，记录s字符串元素出现的个数，然后遍历t，对哈希表内对应字符键的值做自减操作，若小于0，则返回false。

Code

Complexity

Time Space
O( n ) O( n )

Approach#2 Use Set

Solution

205.Isomorphic Strings

From：205. Isomorphic Strings

Description

Given two strings s and t, determine if they are isomorphic.

Two strings are isomorphic if the characters in s can be replaced to get t.

All occurrences of a character must be replaced with another character while preserving the order of characters. No two characters may map to the same character but a character may map to itself.

Example

Given "egg", "add", return true.

Given "foo", "bar", return false.

Given "paper", "title", return true.

Note:
You may assume both s and t have the same length.

Input and output analysis

Approach#1 Convert to pattern string by same rule I

Solution

一个比较简单粗暴的方法，我们可以遍历String，按照字符出现的先后顺序对String内的字符进行重新编码，最后比较比较两个字符串编码内容，相同则返回true。

例如，我们设置一个patterCode,按照出现顺序从1开始对String编码：
对于给定字符串egg，add，编码后均为122，返回true；
对于给定字符串foo，bar，前者编码为122，后者为012，返回false；
对于给定字符串paper，title，编码后均为12134，返回true。

Code

Complexity

Time Space
O( n ) O( n )

Approach#2 Convert to pattern string by same rule II

Solution

设置一个patternNum变量与两个映射表。从前往后遍历s与t，对于每一次遍历在s和t相同位置上的字符做如下操作：

  • 若字符第一次出现则设置相同patterNum，分别记录在两个映射表中。
  • 比较两个映射表中对应字符的patternNum，若不相同则返回false。

Code

Complexity

Time Space
O( n ) O( n )

Note

1.两种String遍历方式：


在循环体内的代码不需要依赖数组索引的情况下，方式二遍历方法会比方式一更高效。

2.根据字符的ASCII码值建立映射数组，这比使用集合框架中的HashMap更高效。
