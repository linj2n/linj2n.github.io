CSS 三大核心模型

Posted on 2018/06/01

CSS 盒模型、定位模型以及浮动模型。

功能

CSS 指层叠样式表。

HTML 标签被设计用于定义文档内容，CSS 为了解决内容与表现分离的问题。

多重样式允许同一个 HTML 文档内部引用多个外部样式表。层叠次序按照从小到大依次排列如下：

  • （1）内联样式（在 HTML 元素内部）
  • （2）内部样式表（位于 标签内部）
  • （3）外部样式表
  • （4）浏览器缺省设置

盒模型

概念

Margin、Padding、Width与Height

  • Margin ： 元素边框外的空白边，一般是透明的，用于控制元素之间的间隔。
  • Padding： 从边框开始到元素内容区域之间的填充，用于内容与背景的分隔。
  • Width 与 Height： 元素内容区域的宽度与高度。

Margin叠加

空白边（Margin）叠加是指两个垂直的空白边相遇时，将形成一个新的空白边，长度为这两个空白边较大者，

只有普通文档流中边框的垂直空白边才会产生空白边叠加，行内框（inline）、浮动框（float）或绝对定位框（absolute）之间的空白便不会叠加。

Block 与 Inline

  • p、h1 或 div等元素被称为块级元素，它们从上往下排列；strong 和 span 等元素被称为行内元素，它们显示在行中从左向右，水平排列。
  • 块级元素之间的垂直距离由 Margin决定，行级元素的水平间距由padding、width与Margin共同决定。
  • display 属性可以改变生成的框的类型。

定位

相对定位

若对一个元素进行相对定位，它将以默认位置（普通文档流中的位置）作为起点，“相对于”这个起点进行垂直与水平位置移动。

但无论是否进行移动，元素会占据空间，因此可能会覆盖其他框。

想要阻止行框（下文的段落）围绕在浮动框的外边，可以对 clear 进行设置，属性值 left、right、both 或 none，表示框的哪些边不挨着浮动框。
