Posted on 2018/04/26

环境配置

安装 mysql

  • 更新Homebrew并安装mysql
  • 若出现Warning: mysql-5.7.10 already installed, it's just not linked错误解决方法：

启动 mysql

  • 使用cd命令进入support-files文件目录，路径为：/usr/local/Cellar/mysql/5.x.x/support-files，执行mysql.server。
  • 可能会出现以下错误
  • 这是由于权限问题造成的，解决办法

配置 mysql

登录 mysql

关闭 mysql

查找 mysql 的进程号，为查询结果的第二列中的数字。

使用 kill 命令结束 mysql，如 mysql 的进程号为 xxxx

导入 mysql-connector jar包到 eclipse中

前往
https://dev.mysql.com/downloads/file/?id=476198
并解压缩，复制mysql-connector-java-5.x.xx.-bin.jar到相应 porject 的WebContent/WEB-INF/lib目录下

右键打开 project 的 properties 属性 -> Java Build Path -> Libraries -> add JARs

选择之前复制的路径 WebContent/WEB-INF/lib 下的 mysql-connector-java-5.x.xx.-bin.jar，点击 OK。

错误解决

解决The server quit without updating PID file (/usr/local/var/mysql/xxxxx.local.pid)

给 mysql 文件夹授权

这是由于权限问题造成的，解决办法

删除 .err 文件

在 mac os 上，mysql 的错误文件存在下面的路径

使用 rm 命令删除即可
