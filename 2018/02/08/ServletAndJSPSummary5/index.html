Servlet/JSP知识小结（5） JSTL 库核心标签

Posted on 2018/02/08

JSP 页面中使用 JSTL 目的是为了不希望出现 Java 业务代码，同时弥补 Html 标签的不足。JSTL 指标准标签库，通过多个标签库来规定其动作指令，其中包括：Core、XML、i18n、Sql、functions 五种标签库，但是除了 Core 标签库，其他的都已经被淘汰了，因此我们只关注 JSTL 中的 Core 标签库。

使用准备

  1. 前往下面的网址
    http://archive.apache.org/dist/jakarta/taglibs/standard/binaries/
    选择最新版本的 zip 包下载并解压，解压后前往 lib 文件夹将 jstl.jar和standard.jar复制到项目 WEB-INF/lib 目录下。

  2. 在需要使用 JSTL 的 JSP 页面头部中插入一下代码，引入 Core 标签库。

Core 标签库分类

Core 标签库根据功能可以分为以下几类

类别 标签 用途
通用动作指令： out 标签、set 标签、remove 标签 主要用于操作各种域变量。
条件动作指令： if 标签、choose 标签、when 标签、otherwise 标签 用于控制流程处理。
iterator 动作指令： forEach 标签、forTokens 标签 用于迭代各种对象或集合。
格式化动作指令： fromatNumber 标签、 用于格式化和解析数字或日期。

通用动作指令

out 标签

out 标签对 EL 表达式进行运算，并将结果输出到当前的 JspWriter 。

语法格式：
out 的语法有一下两种形式，一种有主体内容，另外一种没有，[]为可选属性，可以省略。

使用第一种形式输出Session中的myVar变量的值，其中 EL 表达式获取myVar变量的值。

out 标签的三个属性描述

属性 类型 描述
value Object value= 后面接 EL 表达式。
escapeXml Boolean 规定结果中<，>，'，"和&转换成那种字符实体代码，如<转成&lt;。
defualt Object 默认值

set 标签

语法格式

第一种，设置JavaBean对象属性值，将value赋值给相应 JavaBean 的属性，相当于 setter 方法：target.setProperty(value)。

第二种，设置不同域的相应的变量值，或创建一个域变量，将value赋值给varName变量，varName变量可以是来自page、request、session或application域。

第三种，引用不同于的相应变量，并不做赋值操作，
