HTML 知识小结

Posted on 2018/01/28

概念

名称 解释 示例
标签 HTML是由一套标记标签 （markup tag）组成，通常就叫标签，标签由开始标签和结束标签组成
元素 元素指的是从开始标签到结束标签之间所有的代码，属性用来修饰标签的 <h1 align="center">居中标题</h1>
标题 标题会自动粗体，大写其中的内容，并且带有换行效果 <h1>标题1</h1>
段落 <p>标签即表示段落是paragraph的缩写，自带换行效果 <p>段落4 </p>
粗体 <b> <strong>b是bold的缩写，仅仅表示该文本是粗体的，并不暗示这段文字的重要性。strong虽然也是粗体，但是更多的是强调语义上的加重，提醒用户该文本的重要性。在SEO（搜索引擎优化）的时候，也更加容易帮助用户找到重点的内容 <b>，<strong>
斜体 <i>和<em>都可以表示斜体效果，i是italic的缩写，仅仅表示该文本是斜体的，并不暗示这段文字的重要性，em 是 Emphasized的缩写，虽然也是斜体，但是更多的是强调语义上的加重，提醒用户该文本的重要性。 常常用于引入新的术语的时候使用。
预格式 有时候，需要在网页上显示代码，比如java代码就需要用到<pre>
下划线 <ins>即下划线标签，但是通常来讲，不要给普通文本加下划线，因为用户会误以为是一个超链
删除线 <del>即删除标签，delete的缩写

标签

图像

即图像标签，需要设置其属性 src指定图像的路径。

/ 示例 备注
绝对路径 <img src="file://c:/example.png"/>
相对路径 <img src="../../example.png">
链接 <img src="www.example.com/example.png"/>
设置图片大小 <img width="200" height="200" src="linj2n"/>
图像居中 <div align="right"><img src="../../example.png"></div> img不能够自己居中，需要放在其他能够居中的标签中实现这个效果，比如h1标签,p标签. 经常采用的手段是放在div中居中

超链

/ 示例 备注
<a>显示超链 完整元素： <a href="跳转到的页面地址">超链显示文本</a>
在新的页面打开超链 <a href="http://linj2n.cn" target="_blank">linj2n.cn</a>
超链上的提示文字 <a href="http://linj2n.cn" title="跳转到"打开linj2n.cn" target="_blank">linj2n.cn</a>
使用图片作为超链 <a href="http://www.example.com"><img src="www.example.com/example.png"/></a>

表格

<tr>(table row)表示行，td(table data)表示列

3行2列表格，不带边框

3行2列表格，带边框

设置table宽度

设置table的属性 width，px即像素的意思。例如你的分辨率是1024X768,则你的屏幕横向就有1024个像素。

单元格宽度绝对值

设置td的属性width，在示例里，1单元格设置了宽度，那么3，和a单元格，自动继承该宽度，2单元格的宽度由table宽度和1单元格的宽度决定

单元格宽度相对值

单元格水平对齐

属性align,值[left, center, right]

单元格垂直对齐

属性valign,值[top,middle,bottom]

合并

/ 解释
横跨两列, 水平合并 属性colspan,值1，2，3..
横跨两行, 垂直合并 属性rowspan,值1，2，3..

垂直合并示例

背景色

属性bgcolor

列表

列表分无序列表和有序列表,分别用<ul>标签和<ol>标签表示

块

<div> ,<span>这两种标签都是布局用的,本身没有任何显示效果,通常是用来结合css进行页面布局.

区别：

  • <div>是块元素，即自动换行，常见的块元素还有<h1>,<table>,<p>
  • <span>是内联元素，即不会换行，常见的内联元素还有<img>,<a>,<b>,<srtong>

字体

使用<font>标签表示字体，<font>常用的属性有color,size，颜色表示方法有两种：

  • 颜色名,如black，green
  • 颜色对应的16进制，颜色速查表

内联框架

<iframe> 即内联框架,通过内联框架 可以实现在网页中“插入”网页

表单元素

/ 示例 备注
最简单的文本框 <input type="text">
设置文本框的长度 <input type="text" size="10"> size
为文本框设置初始值 <input type="text" value="有初始值的文本框">双击选中所有代码 value
设置背景文字的文本框 <input type="text" placeholder="请输入账号">
密码框 <input type="password">

单选框

/ 示例 备注
单选框 单选1 <input type="radio" > 单选2 <input type="radio" >
默认选中 默认选中 <input type="radio" checked="checked" > checked="checked"

对单选框进行分组

分组即，多个单选框都在一个分组里，同一时间，只能选中一个单选框，设置name属性相同即可

复选框

下拉列表

<select>即下拉列表，需要配合<option>使用

/ 示例 备注
设置高度 <select size="2"></select>
设置可以多选 <select size="3" multiple="multiple"></select> 使用ctrl或者shift进行多选
默认选中 <option selected="selected">Coding</option><option

文本域

<textarea>即文本域与文本框不同的是，文本域可以有多行，并且可以有滚动条，使用属性cols和rows设置宽度与行数。

按钮

/ 示例 备注
普通按钮 <input type="button" value="一个按钮"> 不可以提交表单
提交按钮 <input type="submit" value="登陆">
重置按钮 <input type="reset" value="重置">
使用图像按钮 <input type="image" src="http://example.com/example.gif">

Q: <input type="submit">与<button>的区别：
A: 只是个单纯的按钮，触发一些画面动作，请使用<button>；反之，会把画面的数据提交给Server的，一般用<input type="submit" />，当然<button>+JS也完全可以取代。
