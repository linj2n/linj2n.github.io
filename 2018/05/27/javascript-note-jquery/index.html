JavaScript 学习笔记之 jQuery

Posted on 2018/05/27

本文是 jQuery 库学习笔记，主要内容有选择器、操作 DOM、AJAX等。

jQuery的优点

  • （1）消除浏览器差异
  • （2）简洁的操作 DOM 的方法
  • （3）轻松实现动画、修改 CSS等各种操作

有关 $

jQuery 把所有的功能全部封装在一个全局变量 jQuery 中，而$是一个jQuery的别名。

选择器

jQuery 的选择器就是帮助我们快速定位到一个或多个 DOM 节点。

DOM 结点与 jQuery对象

  • 查找以#号开头的 DOM 结点，返回的对象是 jQuery 对象，包含这个 DOM 结点所有子结点。
  • jQuery 对象与 DOM 对象之间可以互相转化
  • 通常情况下不需要获取 DOM 对象，直接使用 jQuery 对象即可。

组合查找

将上述简单选择器组合起来进行查找称为组合查找。

在上述按照属性查找属性name=email的例子中，可能会把<div name="email">也查找出来，但是我们只需要<input>类型。此时我们可以使用组合查找，只定位到<input>类型。

层级选择器

DOM 被浏览器解析后是一颗树，表现为层级结构。因此我们很方便的通过 DOM 结点之间的层级结构来定位一个 DOM。

规则： $('ancestor descendant') 层级之间使用空格隔开，层级关系为祖先与后代的关系。

例子

1. 选出 lang-javascript

  • $('ul.lang li.lang-javascript');

2. 选择所有的<li>

  • $('ul.lang li');

子选择器

规则：$('parent>child')，与层级选择器不同，这里的层级关系必须是父子关系。

选出 lang-javascript

  • $('ul.lang>li.lang-javascript');

过滤器

过滤器一般附加在选择器上，能帮助我们更加精确地定位元素。

1. 仅选出JavaScript

  • $('ul.lang li:firstchild');

2. 仅选出Lua

  • $('ul.lang li:lastchild');

3. 选出第N个元素，N从1开始

  • $('ul.lang li:nth-child(2)');

4. 选出序号为偶数的元素

  • $('ul.lang li:nth-child(even)');

5. 选出序号为奇数的元素

  • $('ul.lang li:nth-child(odd)');

表单相关的选择器

选择器 描述
:input 选择<input>,<textarea>,<select>,<button>
:file 选择<input type="file">，等价于input[type=file]
:checkbox 选择复选框，等价于input[type=checkbox]
:radio 选择单选框，等价于input[type=radio]
:focus 选择当前输入焦点的元素，例如把光标放到一个上，用$(‘input:focus’)就可以选出；
:checked 择当前勾上的单选框和复选框，用这个选择器可以立刻获得用户选择的项目，如$(‘input[type=radio]:checked’)；

示例

1. 选择所有的语言

  • selected = $('.test-selector .test-lang');

2. 选择所有动态语言

  • selected = $('ul.test-lang li');

3. 所有静态语言

  • selected = $('ol.test-lang li');

查找与过滤方法

除了使用选择器定位 DOM 结点外，还可以使用 jQuery 提供的查找与过滤方法。一般而言，我们通常先使用选择器定位到一个 jQuery 对象，然后再使用查找与过滤方法。

查找方法

  • （1）find()，在当前结点的所有子节点中查找。
  • （2）parent()，从当前结点开始向上查找。
  • （3）next()与prev()，查找当前结点的同一层级的结点

过滤方法

  • （1）filter()，可以过滤掉不符合选择器条件的结点，或传入一个参数
  • （2）map()，把一个 jQuery 对象包含的若干 DOM 节点转化为其他对象，

操作DOM

通过选择器与 jQuery 提供的方法，我们可以获取到需要的对象，接下来对 DOM 对象进行操作。

更新 DOM

修改 DOM 有以下几种操作：

  • 修改 DOM 文本
  • 修改 DOM 的 CSS
  • 隐藏或显示 DO

下面逐一使用例子介绍其用法。

修改文本

修改 DOM 文本主要通过 jQuery 对象的两个属性：

  • （1）text()，结点的文本
  • （2）html()，结点的原始 HTML 文本

例子

1. 将第一个列表项<li class="js">JavaScript</li>修改为'<span style="color: red">JavaScript</span>')

2. 将第二列表项内容修改为JavaScript & ECMAScript

修改 CSS

修改一个 DOM 的 CSS 有如下方法：

  • $(selector).addClass(class)
  • $(selector).css()

例子

1. 高亮所有 lang dy DOM 对象

  • 方法一：$('#test-css').li.dy>span').css('background-color','#ffd351').css('color','red');
  • 方法二：$('#test-css').li.dy>span').addClass('hightlight');

显示和隐藏 DOM

  • $(selector).hide(speed,callback);

  • $(selector).show(speed,callback);

添加 DOM

  • html()，直接修改
  • append()，可以传入原始的 DOM 对象，jQuery对象和函数对象
  • after()与before()方法，可以用于同级结点

删除 DOM

  • remove()方法

事件

当一个页面加载的同时时，页面上所有的 JavaScript 代码也会被加载。所有加载工作完成后，相关的 JavaScript 代码只能依赖事件来触发了。

常见的事件见下表：

Mouse Events Keyboard Events Form Events Document/Window Events
click keypress submit load
dblclick keydown change resize
mouseenter keyup focus scroll
mouseleave blur(失去焦点) unload

绑定与解绑

绑定方法有二：

  • on方法签名：selector.on(eventsName,function())
  • selector.eventsName(function())

两者是等价的。

鼠标事件

事件 描述
click 鼠标单击时触发；
dblclick 鼠标双击时触发；
mouseenter 鼠标进入时触发；
mouseleave 鼠标移出时触发；
mousemove 鼠标在 DOM 内部移动时触发；
hover 鼠标进入和退出时触发两个函数，相当于 mouseenter 加上 mouseleave。

键盘事件

键盘事件仅作用在当前焦点的 DOM 上，通常是<input>和<textarea>。

事件 描述
keydown 键盘按下时触发；
keyup 键盘松开时触发；
keypress 按一次键后触发。

其他事件

事件 描述
focus 当 DOM 获得焦点时触发；
blur 当 DOM 失去焦点时触发；
change 当<input>、<select>或<textarea>的内容改变时触发；
submit 当<form>提交时触发；
ready 当页面被载入并且 DOM 树完成初始化后触发。
