JavaScript 学习笔记之 操作 DOM

Posted on 2018/05/27

本文是 JavaScript 语言的学习笔记，结合若干个例子介绍了如何使用 JavaScript 对 DOM 结点进行操作，包括对控件与表单的操作。

操作DOM

DOM 是一个棵代表着 HTML 文档被浏览器解析后生成的树，操作一个 DOM 结点实际上有这么几个操作：

  • 更新：改变该DOM结点表示的HTML内容
  • 遍历：遍历该DOM结点下的子结点，以便下一步操作
  • 添加：在该DOM结点下增加一个子结点，相当于动态增加一个HTML结点
  • 删除：删除该DOM结点的内容以及它包含的所有子结点

获取 DOM

要想对 DOM 结点进行操作，前置操作是获取一个 DOM 结点。有两种方式获取 DOM 结点：

  • getElementByXXx()方法，XXx可以是id、tagName或className
  • querySelector()与querySelectorAll方法，需要了解selector语法规则

插入 DOM

在 DOM 结点内插入新的 DOM ，方法有二：

  • appendChild()方法
    • 直接使用appendChild()，将子结点添加到父节点的最后一个子结点
    • 查找子节点的后，使用appendChild()，在指定位置插入一个子结点
  • parentElement.insertBefore(newElement, referenceElement)子节点插入到referenceElement之前。

删除 DOM

获取该结点点的父节点，调用父节点的removeChild删除该结点。

注意到删除后的节点虽然不在文档树中了，但其实它还在内存中，可以随时再次被添加到别的位置。

操作表单

表单控件

HTML 表单输入控件常见的有以下几种：

  • 文本框，<input type="text">，用于输入文本；
  • 口令框，<input type="password>，用于输入密码；
  • 单选框，<input type="radio">，用于选择一个项
  • 复选框，<input type="checkbox">，用于选择多项；
  • 下拉框，<select>，用于选择一个项；
  • 隐藏文本，<input type="hidden">，用户不可见，但表单提交的时候会把隐藏文本发送到服务器

HTML5 新增了大量标准空间，均使用<input>标签。

  • date
  • datetime
  • datetime-local
  • color

支持HTML5的浏览器将获得格式化的字符串。例如，type="date"类型的 input 的 value 将保证是一个有效的 YYYY-MM-DD 格式的日期，或者空字符串。

获取控件值

两个步骤可以获取控件值：

  • （1）获取<input>结点的引用
  • （2）调用value获得对应用户输入值。

以上两个步骤可以用于获取所有控件的值，但通过checkbox与radio的value属性返回的永远的是预设值。

若想要获得用户对checkbox与radio的值，则需要使用它们的checked属性值判断。

设置控件值

与获取控件值类似，直接对value进行赋值。
